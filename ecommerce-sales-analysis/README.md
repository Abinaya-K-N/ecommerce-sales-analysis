# E-commerce Sales Analysis

## Project Overview
This project performs a complete data analysis on e-commerce sales data to identify revenue trends, category performance, and monthly sales behavior using Python.

## Objectives
- Load and clean e-commerce sales data
- Analyze revenue and sales trends
- Create professional visualizations
- Generate meaningful business insights

## Technologies Used
- Python
- Pandas
- Matplotlib

## Project Structure
- data/: Contains the dataset
- visualizations/: Stores generated charts
- report/: Contains written insights
- main.py: Main analysis script

## Setup Instructions
1. Install dependencies:
   pip install -r requirements.txt
2. Run the project:
   python main.py

## Output
- Bar chart showing sales by product category
- Line chart showing monthly sales trends
- Pie chart showing revenue distribution
