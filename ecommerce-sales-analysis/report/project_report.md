## Dataset Description
The dataset contains e-commerce transaction data including transaction date, product category, and purchase amount. The data is used to analyze sales performance and revenue trends.

## Analysis Performed
1. Category-wise sales analysis using bar charts
2. Monthly sales trend analysis using line charts
3. Revenue distribution analysis using pie charts

## Key Insights
1. Certain product categories contribute significantly more revenue than others.
2. Monthly sales trends show variations that help identify peak and low-performing periods.
3. Revenue distribution highlights dependency on top-performing product categories.

## Business Impact
This analysis helps businesses understand sales patterns, optimize inventory planning, and make informed decisions based on revenue trends.
